# Clear Purple

A dark Obsidian theme built on a GitHub-dark base (`#0d1117`) with a vibrant purple heading hierarchy and comfydark-sourced code syntax colors.

## Design

- **H1** — bordered box with a soft purple glow, acts as a major section break
- **H2–H6** — uppercase labels with a left border, decreasing in size and intensity down the hierarchy
- **Code blocks** — comfydark syntax palette (orange strings, blue keywords, teal properties)
- **UI shell** — sidebar, tabs, ribbon, scrollbars, and focus states all themed in matching purple

## Installation

### From Obsidian community themes
Search for **Clear Purple** in Settings → Appearance → Themes.

### Manual
1. Download `theme.css` and `manifest.json`
2. Copy both into `.obsidian/themes/Clear Purple/` inside your vault
3. Enable the theme in Settings → Appearance
